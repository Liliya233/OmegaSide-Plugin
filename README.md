# OmegaSide插件
**简要介绍**

    适用于 OmegaSide旁加载组件系统 的插件，或许能为你带来一些帮助

**运行方式**

    方式一：使用 VSCode 打开文件夹，然后直接运行对应的插件 (需要先运行Omega)

    方式二：把插件放到 omega_python_plugins 目录下，然后运行Omega

    ...

**注意事项**

    1. 插件都依赖于 API_By_Liliya.py 运行，请确保它们处于同一目录下
    
    2. 若想应用这些插件，你可能需要自行修改代码进行适配
    
    3. 未经允许，禁止商用！
